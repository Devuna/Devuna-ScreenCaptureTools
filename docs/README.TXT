Using Screen Capture Tools for Clarion 10.0:

Documentation can be found in SCTOOLS.CHM located in the SoftVelocity\Clarion10\accessory\Documents\Devuna\ScreenCaptureTools folder.
It is available on the Start Menu as |Clarion 10.0 Accessories|Devuna|ScreenCaptureTools|Help
It is available on the Clarion10 IDE menu |Help|Accessory|Devuna|ScreenCaptureTools

An example application is provided in the C:\Users\Public\Documents\SoftVelocity\Clarion10\Accessory\Devuna\Examples\ScreenCaptureTools folder.
It is available on the Start Menu as |Clarion 10.0 Accessories|Devuna|ScreenCaptureTools|Example
It is available on the Clarion 10 IDE menu |Help|Accessory|Devuna|Examples in the ScreenCaptureTools folder.

You must add the SCT_Initialization global extension template to your application
before using any library calls or templates.

The following files are required to use the Screen Capture Tools library:

DIBAPI.DLL
IJL15.DLL
JPGAPI.DLL
SCTOOL100.DLL
