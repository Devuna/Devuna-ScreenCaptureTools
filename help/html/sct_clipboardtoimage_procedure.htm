SCT_ClipboardToImage Procedure (paste clipboard into image control)

SCT_ClipboardToImage( LONG param_hWindow, LONG param_feq )

SCT_ClipboardToImage  Pastes the Windows clipboard contents into an image control. The clipboard must contain data in one of the following formats:

  • CF_DIB
  • CF_BITMAP
  • CF_ENHMETAFILE
  • CF_METAFILEPICT
param_hWindow
The window handle of the window containing the image control
param_feq
The field equate number for the image control

Return Value   0 if successful
 or, one of these symbolic constants, defined in sctools.inc, if not successful.

  • ERR_NOTINITIALIZED
  • ERR_FORMATNOTAVAILABLE
