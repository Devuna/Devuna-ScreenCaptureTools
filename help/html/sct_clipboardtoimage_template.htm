SCT_ClipboardToImage code template

The SCT_ClipboardToImage template generates code to paste the contents of the clipboard into an image control.

Image Control  Select the image control from the list. The list contains all image controls in scope for this procedure

Check for Error  This check box controls the generation of error checking code. If an error is returned from the Screen Capture Tools library, and error message will be displayed.

This template generates code similar to the following:

SCT_ClipboardToImage(Window{PROP:Handle},?Image1)
