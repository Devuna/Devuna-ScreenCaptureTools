SCT_Error Procedure (display an error message)

SCT_Error(USHORT param_wErrNo )

SCT_Error  Displays a Message with an error message corresponding to the specified error number.

param_wErrNo  parameter must be set to one of the following:

ERR_NOT_DIB
Tried to load a non-DIB file.
ERR_MEMORY
Not enough memory.
ERR_READ
Error reading a file.
ERR_LOCK
Error on a GlobalLock.
ERR_OPEN
Error opening a file.
ERR_CREATEPAL
Error creating a palette.
ERR_GETDC
Error getting a DC.
ERR_CREATEDDB
Error creating a bitmap.
ERR_STRETCHBLT
StretchBlt failed.
ERR_STRETCHDIBITS
StretchDIBits failed.
ERR_SETDIBITSTODEVICE
SetDIBitsToDevice failed.
ERR_STARTDOC
Error calling StartDoc.
ERR_NOGDIMODULE
GDI module was not in memory.
ERR_SETABORTPROC
Error calling SetAbortProc.
ERR_STARTPAGE
Error calling StartPage.
ERR_NEWFRAME
Error calling NEWFRAME escape.
ERR_ENDPAGE
Error calling EndPage.
ERR_ENDDOC
Error calling EndDoc.
ERR_SETDIBITS
Error calling SetDIBits.
ERR_FILENOTFOUND
Error opening file in LoadDIBLoadDIB.
ERR_INVALIDHANDLE
Invalid handle.
ERR_DIBFUNCTION
Error on call to DIB function.
ERR_NOTINITIALIZED
Screen Capture Tools library not initialized.
ERR_FORMATNOTAVAILABLE
Clipboard does not contain a supported format.

Return Value   None.
