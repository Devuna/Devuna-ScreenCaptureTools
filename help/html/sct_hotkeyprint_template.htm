SCT_HotKeyPrint extension template

The SCT_HotKeyPrint application global extension template provides print screen hot key activation to an application. The template also provides an option for a popup menu to allow the user to print, save, or copy the captured image to the clipboard

Window to Print  Select one of the choices from the dropdown list. The choices and their meaning are as follows:

This Window
the current procedure window
Top Window
the first child window for the application
Application
the application window
DeskTop
the desktop window

Print Option  Select one of the choices from the dropdown list. The choices and their meaning are as follows:

Best Fit
Fits the printed screen on the page in as large of an area as possible while still retaining the correct screen aspect ratio. The param_wScaleX and param_wScaleY parameters are not used if this option is selected.
Stretch to Page
Stretches the printed screen to fill the entire printed page. The param_wScaleX and param_wScaleY parameters are not used if this option is selected.

Scale  Scales the printed screen by the integer scaling factors specified in the param_wScaleX and param_wScaleY parameters. If the scaling is set too large, the screen may extend off the printed page and will be clipped.

X Scale  Specifies the x-axis scaling factor for the printed window. The x and y scaling factors are only used if the Scale option above is selected.

Y Scale  Specifies the y-axis scaling factor for the printed window.

Hot Key  Specifies the hot key to be used.

Use popup menu  If this option is enabled, a popup window will be presented to the user to allow them to print the captured image to the printer, save the captured image to a disk file, or copy the captured image to the clipboard.

Compress Image If this option is enabled, images saved to disk will be in JPEG format. If this option is not enabled, images saved to disk will be in DIB format.
