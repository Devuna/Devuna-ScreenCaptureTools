SCT_Init Procedure (initialize the screen capture tools library)

SCT_Init( LONG param_hWndApplication, LONG param_hWndClient, LONG param_hInstance )

SCT_Init  Initializes the Screen Capture Tools Library. This procedure must be called before making any other calls to the library.

param_hWndApplication
The window handle for the application main window.
e.g. AppFrame{PROP:Handle}
param_hWndClient
The window handle for the application client window.
e.g. AppFrame{PROP:ClientHandle}
param_hInstance
The instance handle for the .EXE file.
e.g. SYSTEM{PROP:AppInstance}

Return Value   None.

See Also:   SCT_Initialization Template
