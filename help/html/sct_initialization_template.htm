SCT_Initialization extension template

This template adds the necessary equates and libraries to your application to enable you to utilize the Screen Capture Tools API calls. The template also adds code to the First Procedure (as specified in the Application Properties dialog) to initialize the Screen Capture Tools Library.

Generate template globals as EXTERNAL

Adds the EXTERNAL attribute to the global variable declarations generated by the templates. This means your program relies on an external library to allocate memory for these variables, and to export them so your program can access them.

The templates do not currently generate any global variable declarations. This option is reserved for future use.

External Globals and ABCs Source Module

Specify whether the external library is dynamically or statically linked.

This sets the flag parameter of the DLL attribute for template generated class declarations. See the Language Reference for more information on the DLL attribute.

This template generates code similar to the following:

SCT_Init(AppFrame{PROP:Handle},AppFrame{PROP:ClientHandle},SYSTEM{PROP:AppInstance})
