SCT_PrintClipboard code template

The SCT_PrintClipboard template generates code to print the clipboard contents to the default printer.

Print Option

Select one of the choices from the dropdown list. The choices and their meaning are as follows:

Best Fit
Fits the printed screen on the page in as large of an area as possible while still retaining the correct screen aspect ratio. The param_wScaleX and param_wScaleY parameters are not used if this option is selected.
Stretch to Page
Stretches the printed screen to fill the entire printed page. The param_wScaleX and param_wScaleY parameters are not used if this option is selected.
Scale
Scales the printed screen by the integer scaling factors specified in the param_wScaleX and param_wScaleY parameters. If the scaling is set too large, the screen may extend off the printed page and will be clipped.

X Scale  Specifies the x-axis scaling factor for the printed window. The x and y scaling factors are only used if the Scale option above is selected.

Y Scale  Specifies the y-axis scaling factor for the printed window.

X Position  Specifies the x-axis origin for the printed window.

Y Position  Specifies the y-axis origin for the printed window.

Check for Error  This check box controls the generation of error checking code. If an error is returned from the Screen Capture Tools library, and error message will be displayed.

Result Field  Select the local data variable that will store the result of the library call. Default is SCT_wResult.

The template automatically creates a local data variable for the result field.

This template generates code similar to the following:

SCT_wResult = SCT_PrintClipboard(ThisWindow{PROP:Handle}) !print clipboard contents
IF SCT_wResult                                            !if an error is returned
   SCT_Error(SCT_wResult)                                 !   display error message
END                                                       !endif
