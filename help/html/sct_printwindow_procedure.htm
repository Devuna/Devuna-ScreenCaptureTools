SCT_PrintWindow Procedure (capture window to default printer)

SCT_PrintWindow( UNSIGNED param_hWindow [, LONG param_printArea=PW_WINDOW] [, LONG param_printOption=PW_BESTFIT] [, LONG param_wScaleX=1] [, LONG param_wScaleY=1] [, LONG param_x=0] [, LONG param_y=0] )

SCT_PrintWindow  Prints the window associated with the window handle to the default printer.

param_hWindow
The window handle of the window to be printed.
param_printArea
An integer constant or variable that specifies the area of the window to be printed. See below for a listing of the possible values for this parameter. If omitted, the entire window is printed.
PW_WINDOW
Print the entire window
PW_CLIENT
Print the client area of the window
param_printOption
An integer constant or variable that specifies the printing options.
PW_BESTFIT
Fits the printed screen on the page in as large of an area as possible while still retaining the correct screen aspect ratio. The param_wScaleX and param_wScaleY parameters are not used if this option is selected. This is the default value.
PW_STRETCHTOPAGE
Stretches the printed screen to fill the entire printed page. The param_wScaleX and param_wScaleY parameters are not used if this option is selected.
PW_SCALE
Scales the printed screen by the integer scaling factors specified in the param_wScaleX and param_wScaleY parameters. If the scaling is set too large, the screen may extend off the printed page and will be clipped
param_wScaleX
Specifies the x-axis scaling factor for the printed window. The x and y scaling factors are only used if the PW_SCALE option below is selected.
param_wScaleY
Specifies the y-axis scaling factor for the printed window.
param_x
Specifies the x-axis origin for the printed window.
param_y
Specifies the y-axis origin for the printed window.

Return Value   0 if successful
 or, one of these symbolic constants, defined in sctools.inc, if not successful.

  • ERR_NOTINITIALIZED
  • ERR_DIBFUNCTION
  • ERR_INVALIDHANDLE
  • ERR_LOCK
  • ERR_SETABORTPROC
  • ERR_STARTDOC
  • ERR_NEWFRAME
  • ERR_ENDDOC
  • ERR_GETDC
  • ERR_STRETCHDIBITS
