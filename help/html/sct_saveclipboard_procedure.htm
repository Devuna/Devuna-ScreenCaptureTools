SCT_SaveClipboard Procedure (save clipboard contents to file)

SCT_SaveClipboard( UNSIGNED param_hWindow, *CSTRING param_szFilename [, BYTE param_Compress=0] )

SCT_SaveClipboard  Saves the clipboard to the specified filename in either DIB or JPEG format. Although the convention for naming DIB files is to use a .BMP extension and the convention for naming JPEG files is to use a .JPG extension, the entire file name must be specified.

param_hWindow
The window handle of the calling window.
param_szFilename
A string variable that specifies the complete path name of the DOS filename under which the clipboard contents are to be saved.
param_Compress
A byte variable that determines the saved file format. 0 = DIB and 1 = JPEG.

Return Value   0 if successful
 or, one of these symbolic constants, defined in sctools.inc, if not successful.

  • ERR_NOTINITIALIZED
  • ERR_INVALIDHANDLE
  • ERR_OPEN
  • ERR_LOCK
  • ERR_FORMATNOTAVAILABLE
