SCT_SaveSelectedArea Procedure (save selected area to bitmap file)

SCT_SaveSelectedArea( *CSTRING param_szFilename [, BYTE param_Compress=0] )

SCT_SaveSelectedArea  Allows the user to select an area on the screen then saves the selected area to the clipboard or specified filename in DIB format. The convention for naming DIB files is to use a .BMP extension, the entire file name must be specified.

param_szFilename
A string variable that specifies the complete path name of the DOS filename under which this bitmap (DIB) is to be saved. If the param_szFilename value is "clipboard" the bitmap is saved to the windows clipboard.
param_Compress
A byte variable that determines the saved file format. 0 = DIB and 1 = JPEG.

Return Value   0 if successful
 or, one of these symbolic constants, defined in sctools.inc, if not successful.

  • ERR_NOTINITIALIZED
  • ERR_INVALIDHANDLE
  • ERR_OPEN
  • ERR_LOCK
