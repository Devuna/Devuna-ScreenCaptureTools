SCT_SaveWindow Procedure (save window to clipboard or bitmap file)

SCT_SaveWindow( UNSIGNED param_hWindow, *CSTRING param_szFilename [, LONG param_printArea=PW_WINDOW] [, BYTE param_Compress=0] )

SCT_SaveWindow  Saves the desired window to the clipboard or specified filename in DIB format. Although the convention for naming DIB files is to use a .BMP extension, the entire file name must be specified.

param_hWindow
The window handle of the window to be saved.
param_szFilename
A string variable that specifies the complete path name of the DOS filename under which this bitmap (DIB) is to be saved. If the param_szFilename value is "clipboard" the bitmap is saved to the windows clipboard.
param_printArea
An integer constant or variable that specifies the area of the window to be printed. See below for a listing of the possible values for this parameter. If omitted, the entire window is printed.
param_printArea
parameter must be set to one of the following:
PW_WINDOW
Print the entire window. This is the default value.
PW_CLIENT
Print the client area of the window
param_Compress
A byte variable that determines the saved file format. 0 = DIB and 1 = JPEG.

Return Value   0 if successful
 or, one of these symbolic constants, defined in sctools.inc, if not successful.

  • ERR_NOTINITIALIZED
  • ERR_INVALIDHANDLE
  • ERR_OPEN
  • ERR_LOCK
